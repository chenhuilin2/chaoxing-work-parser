## 油猴脚本：一键提取学习通题目

不用再手动复制粘贴题目。该脚本提供一键提取学习通题目功能（支持脚本猫/Tampermonkey）。


### 安装
1. 确保已安装脚本管理器（[脚本猫](https://docs.scriptcat.org/) 或 Tampermonkey）
2. 打开管理器面板，点击 **"新建"** 或 **"添加脚本"**
3. 将 `学习通题目提取器.user.js` 文件内容粘贴进去，保存


### 快速开始
1. 打开学习通作业页面（确保题目已加载）
2. 页面右侧会出现 **「提取题目」** 浮动按钮，点击展开面板
3. 点击 **「提取本页题目」**，查看统计结果
4. 如需附加答案，勾选 **「附加答案」**，答案汇总会附在文件末尾
5. 点击 **「下载 .txt」** 或 **「复制文本」**


### 提取原理
- 通过 DOM 选择器 `mark_item` / `questionLi` 识别题目容器
- 自动提取题干、选项、正确答案
- 输出格式与手动复制粘贴一致
