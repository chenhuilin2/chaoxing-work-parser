## Chaoxing Work Parser — 学习通作业题目一键提取，快速生成一份 Word 试卷

这个脚本可以让你不用再手动复制粘贴超星学习通的题目。只需在脚本猫插件中新建脚本，把代码复制上去，就可以一键提取学习通作业题目并导出 **Word 试卷**（.docx），方便进行期末复习。

导出支持格式：**Word 试卷** / TXT / Markdown。


### 安装
1. 确保已安装脚本管理器（[脚本猫](https://docs.scriptcat.org/)）
2. 打开安装页面 ([学习通作业提取器](https://scriptcat.org/zh-CN/script-show-page/6713))


### 使用
1. 进入你想提取的学习通作业页面
2. 点击页面右侧 **「提取题目」** 浮动按钮，展开面板
3. 点击 **「提取本页题目」**，可查看题目统计
4. 选择输出格式：
   - **Word 试卷** — 直接导出 .docx 试卷，排版规范，支持乱序输出
   - **TXT** — 纯文本
   - **MD** — Markdown 格式
5. 点击 **「下载」** 或 **「复制文本」**


### 输出格式说明

| 格式 | 适用场景 | 答案/错题 |
|------|---------|----------|
| Word 试卷 | 打印复习 | 不支持（纯试卷） |
| TXT | 归档、对照复习 | 支持 |
| Markdown | 导入笔记软件 | 支持 |


### 提取原理
- 通过 DOM 选择器 `mark_item` / `questionLi` 识别题目容器
- 自动提取题干、选项、正确答案
- Word 试卷使用纯前端 docx 库生成，无需后端
- 输出格式与手动复制粘贴一致
