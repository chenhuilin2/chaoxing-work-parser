## Chaoxing Work Parser — 学习通作业题目一键提取，快速生成一份 Word 试卷

这个脚本可以让你不用再手动复制粘贴超星学习通的题目。只需在脚本猫插件中新建脚本，把代码复制上去，就可以一键提取学习通作业题目并导出 **Word 试卷**（.docx），方便进行期末复习。

导出支持格式：**Word 试卷** / TXT / Markdown。


### 安装
1. 确保已安装脚本管理器（[脚本猫](https://docs.scriptcat.org/)）
2. 打开安装页面 ([学习通作业提取器](https://scriptcat.org/zh-CN/script-show-page/6713))


### 使用
1. 进入你想提取的学习通作业页面（支持"查看答案"和"做作业"两种页面）
2. 点击页面右侧 **「提取题目」** 浮动按钮，展开面板
3. 点击 **「提取本页题目」**，可查看题目统计
4. 选择输出格式（默认 Word 试卷）：
   - **Word 试卷** — 直接导出 .docx 试卷，排版规范，支持乱序输出和附加答案
   - **TXT** — 纯文本
   - **MD** — Markdown 格式
5. 按需勾选选项（附加答案、附加错题、打乱顺序）
6. 点击 **「下载」** 或 **「复制文本」**


### 输出格式说明

| 格式 | 适用场景 | 答案/错题 |
|------|---------|----------|
| Word 试卷 | 打印复习 | 支持 |
| TXT | 归档、对照复习 | 支持 |
| Markdown | 导入笔记软件 | 支持 |


### 提取原理
- 自动识别页面类型，支持两种 DOM 结构：
  - **查看答案页面**：通过 `.mark_item` 容器分题型提取，支持正确答案和错题标注
  - **做作业页面**：通过 `.questionLi` 的 `typeName` 属性识别题型，提取题干和选项
- 自动提取题干、选项、图片（Word 嵌入 base64，TXT/MD 保留 URL）
- Word 试卷使用纯前端 docx 库生成，无需后端
- 输出格式与手动复制粘贴一致
